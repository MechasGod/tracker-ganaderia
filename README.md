## Estructura del Proyecto

Este proyecto está organizado en 3 ramas principales:

## Ramas

1. **documentacion** - Rama dedicada a la documentación del proyecto
2. **fuente** - Rama dedicada al código fuente
3. **pruebas** - Rama dedicada a las pruebas (testing)

## Carpetas

El proyecto incluye 3 carpetas correspondientes a las ramas:

- `documentacion/` - Carpeta para documentación
- `fuente/` - Carpeta para código fuente  
- `pruebas/` - Carpeta para pruebas


## Comandos para trabajar con las ramas

Para cambiar entre ramas:
```bash
git switch documentacion
git switch pruebas
git switch fuente
```

Para subir las ramas al repositorio remoto:
```bash
git push origin documentacion
git push origin fuente
git push origin pruebas
```
